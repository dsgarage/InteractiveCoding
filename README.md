###ColorRect
中心から線を伸ばしてその先に四角を描画。
四角と線をランダムに描画していたが、四角の座標を線の先端にしたら追尾すると考え連動。

###SoloRect
とりあえずひとつの四角で色を変えるということに立ち返るためのスケッチ。
xyそれぞれをランダムにしてしまうとタイムマシンの移動中空間みたいになるので、ちょっと気持ち悪かった
今回はxだけランダムに
